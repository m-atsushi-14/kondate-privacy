kondate（こんだて） プライバシーポリシー

最終更新日: 2026-06-28

kondate（以下「本アプリ」）は、利用者のプライバシーを尊重し、取得する情報を必要最小限にとどめます。本ポリシーは、本アプリが取り扱う情報とその目的を説明します。

1. 取得する情報

  • アカウント情報（メールアドレス）：ログイン・本人確認のために取得します（認証基盤 Amazon Cognito）。
  • 利用者が作成したデータ：レシピ（「うちの料理」）、献立、買い物リスト、食費、タグ等。レシピとプロフィールはサーバー（Amazon DynamoDB）に保存され、献立・買い物リスト・食費は主に端末内に保存されます。
  • 位置情報（任意）：天気に合わせたヒントを表示するためにのみ使用します。許可した場合に限り、現在地の座標を天気サービス（Open-Meteo）へ送信して天気を取得します。座標を当社サーバーに保存することはありません。許可しなくても本アプリは利用できます。
  • AI機能の入力：レシピ生成・献立相談・レシピ検索を使った場合、入力された料理名や相談メッセージを処理のためサーバー（Amazon Bedrock／検索は Tavily）へ送信します。

2. 利用目的

  • 本アプリの機能提供（ログイン、データの保存・同期、買い物リストの作成）
  • AIによる献立提案・レシピ材料の生成・レシピページの検索（利用者が機能を使った場合）
  • 天気に合わせたヒントの表示（位置情報を許可した場合）

3. 第三者への提供・委託

サービス提供に必要な範囲で、以下の外部サービスを利用します。これらの目的以外で個人情報を第三者に販売・提供することはありません。

  • Amazon Web Services（Cognito / DynamoDB / Bedrock）：認証・データ保存・AI処理
  • Tavily：レシピページのリンク検索（送信するのは料理名のみ）
  • Open-Meteo：天気の取得（位置情報を許可した場合の座標）
  • Apple：アプリ配信・課金（将来）

4. 広告・トラッキング

本アプリは広告を表示しません。広告目的のトラッキングや、第三者への行動データの提供は行いません。

5. データの保存と削除

アカウントを削除すると、サーバー上のレシピ・プロフィール、および認証情報は削除されます。端末内のデータはアプリの削除で消去されます。削除のご希望や不明点は下記までご連絡ください。

6. お子様のプライバシー

本アプリは一般向けで、13歳未満の子供を対象として個人情報を意図的に収集することはありません。

7. 本ポリシーの変更

必要に応じて本ポリシーを改定することがあります。重要な変更がある場合はアプリ内またはこのページでお知らせします。
